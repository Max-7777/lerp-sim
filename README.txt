Linear Interpolation Simulator!

Run the .jar with a recent version of java (17 or 20) 

⁍ Click to add dot.
⁍ Right-click to remove dot.
⁍ Use the slider in the bottom right to interpolate the dots.

⁍ 'r' to delete all dots.
⁍ '1-5' to change theme.
⁍ 'f' fps.
